# Sudoku Web Application

A simple HTML-based Sudoku game that runs in any web browser.

## Features

- Interactive 9x9 Sudoku grid
- Pre-loaded puzzle with given numbers
- Real-time validation with visual feedback
- Input restriction to numbers 1-9 only
- Solution checking
- Reset functionality

## How to Play

1. Open `sudoku.html` in any web browser
2. Fill empty cells with numbers 1-9
3. Each row, column, and 3x3 box must contain all digits 1-9
4. Invalid entries are highlighted in red
5. Click "Check Solution" to verify completion
6. Click "Reset" to start over

## Files

- `sudoku.html` - Complete web application (HTML, CSS, JavaScript)
- `README.md` - This documentation

## Browser Compatibility

Works in all modern web browsers including Chrome, Firefox, Safari, and Edge.
